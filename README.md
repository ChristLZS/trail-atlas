# 山野路书 · 发现中国经典徒步路线

> A静态前端 Demo，为 **TRAE AI 创造力大赛**（生活娱乐赛道）打造。

汇集国内经典徒步路线，整合风景图片、路线难度、最佳季节、行程攻略与安全提示，帮助用户更轻松地规划下一次山野之旅。

## 在线预览

👉 https://christlzs.github.io/trail-atlas/

## 核心模块

- **首屏 Hero**：核心卖点 + 山形 SVG 插画 + 数据指标
- **热门路线**：8 条国内经典徒步（武功山 / 雨崩 / 虎跳峡高路 / 冈仁波齐 / 洛克线 / 长穿毕 / 四姑娘山大峰 / 喀纳斯）
- **难度筛选**：全部 / 入门 / 进阶 / 高阶 / 挑战
- **路线详情弹窗**：4 项快览 + 介绍 / 行程 / 装备 / 安全 四个分区
- **徒步安全提示**：6 条核心要点
- **关于项目 + 页脚**

## 本地运行

```bash
# 直接打开即可，无需服务器
open index.html
```

## 技术栈

纯 HTML + CSS + 原生 JavaScript，零依赖。所有路线封面均由 SVG 程序化生成，离线可用。
